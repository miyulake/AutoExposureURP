# Simple URP Auto Exposure
If you use or modify this package, credit is appreciated.

## Requirements
- Unity 2022.3+
- Universal Render Pipeline (URP 14)

## Usage
- Add via [Package Manager](https://docs.unity3d.com/Manual/upm-ui-giturl.html) using Git URL.
```
https://github.com/miyulake/AutoExposureURP.git
```
- Add Renderer Feature to Universal Renderer Data
- Select Auto Exposure
- Change settings to suit your scene
